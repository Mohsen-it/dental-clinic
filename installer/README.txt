WELCOME TO DENTAL CLINIC MANAGEMENT AGORRACODE
==============================================

Thank you for installing Dental Clinic Management AgorraCode, a comprehensive solution for managing dental clinic operations.

SYSTEM REQUIREMENTS
==================
- Windows 10 or later (64-bit recommended)
- Minimum 4GB RAM
- 2GB free disk space
- Internet connection (for updates and license verification)

GETTING STARTED
===============
1. Launch the application from the desktop shortcut or Start menu
2. The application will initialize the database on first run
3. Follow the setup wizard to configure your clinic settings
4. Start managing your dental practice efficiently

KEY FEATURES
============
- Patient Management
- Appointment Scheduling
- Treatment Records
- Inventory Management
- Financial Reports
- Laboratory Orders
- Prescription Management
- Backup and Restore
